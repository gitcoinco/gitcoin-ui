# Banner

<Meta of={BannerStories} />

The Banner component displays a rounded image representing an Banner. The component supports multiple sources, with fallbacks in place if a source fails during runtime.

# Sources (in order)

- No IPFS CID, No URL, and No Fallback Name -> Default Logo
- IPFS CID and an empty or null URL -> ipfsBaseURL + ipfsCid
- URL and an empty or null IPFS CID -> URL
- both an IPFS CID and a URL -> ipfsBaseURL + ipfsCID
